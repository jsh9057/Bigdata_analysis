# Bigdata_analysis

영화 평점과 흥행의 관계를 상관계수로 분석.

개발날짜 2018/03 ~ 2018/07
-------------------------------------------------------------------

데이터 범위 : 개봉 후 3주
	      누적관객수 100만명 이상을 흥행 영화로 간주.
	      흥행영화 집단과 비흥행영화 집단 두 집단으로 나누어 각각 분석.

각 1,2,3 주차 평점을 각 주의 1,2,3 주차 입장객의 수와 상관관계 분석.

결과 : 각 주차 별 평점과 입장객수의 상관관계 및 긍정사전 부정사전
